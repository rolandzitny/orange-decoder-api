Hexstring Decoder - REST API

To use this API, send a POST request with the hexstring as the payload.

Example Using PowerShell (Windows):

Invoke-RestMethod -Uri "http://34.147.166.143:3000/decode/" -Method Post -Headers @{"Content-Type" = "application/json"} -Body '{"hexstring": "cbb2094e01a70109927fff"}'

Example Using curl:

curl -X POST http://34.147.166.143:3000/decode/ -H "Content-Type: application/json" -d "{\"hexstring\": \"cbb2094e01a70109927fff\"}"
